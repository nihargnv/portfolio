VicharX – Intelligent Document Management Platform

Technologies Used

Python

Django

MySQL

Transformers

scikit-learn

HTML/CSS/JS

Docker

VicharX is an AI-driven document management platform built for intelligent storage, classification, and retrieval of digital files. Inspired by enterprise-grade systems, it is engineered to offer smart search and automated document understanding while remaining lightweight enough to run on standard hosting environments.

Core Functionalities

  • Document Classification: Auto-classifies uploads into invoices, certificates, prescriptions, IDs, and contracts using an ensemble classifier with scikit-learn pipelines.
  • Semantic Search Engine: Integrates vector search and traditional query scoring to match by filename, type, content summary, or tag relevance.
  • Summarisation: Extracts concise previews using transformers to allow users to skim contents instantly without opening large documents.
  • Lightweight Interface: Built using pure HTML, CSS, and vanilla JavaScript for speed and responsiveness with zero dependency on frontend frameworks.

Tech Stack & Architecture

  • Backend: Django 4.0 with Django ORM and REST endpoints for document CRUD, classification, and search APIs.
  • Database: MySQL relational schema storing metadata, user sessions, summaries, and document types.
  • ML Services: Python scripts containerised with Docker, executing classification and summarisation asynchronously upon upload.
  • Deployment: Locally containerised with planned extension to Google Cloud using App Engine and Cloud SQL.

Advantages Over Traditional File Managers

  • Zero manual tagging needed due to automated file analysis.
  • Search results ranked by semantic weight, not just text match.
  • Summaries reduce open-and-scroll time significantly for large files.
  • Easy to integrate into existing infrastructure due to stateless APIs and optional webhooks.

Planned Enhancements

  • Drag-and-drop bulk upload with real-time feedback on classification progress.
  • Live collaboration with edit/version control on shared documents.
  • Role-based user permissions for access-controlled environments.
  • Embedded AI assistant for natural language Q&A over document content.

Project Scope

VicharX is being developed as a multi-phase project. The current version functions as a single-user prototype tested with mock datasets. The system is being iteratively enhanced to support multi-user access, audit trails, and full cloud migration by Q3 2025.
