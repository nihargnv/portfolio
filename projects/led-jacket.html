LED Dance Jacket - Wearable Synchronized Lighting

Technologies Used

Arduino Uno

NodeMCU (ESP8266)

WS2812B LEDs

RF Transmitter/Receiver

C++

Power MOSFETs

The LED Dance Jacket is a microcontroller-controlled wearable system designed for performance environments. It enables synchronized lighting patterns on the jacket using programmable addressable LEDs. The system supports both standalone animations and remote triggering for group performances, allowing multiple jackets to flash in perfect sync during dance routines or stage shows.

Implemented Functionalities

  • LED Segmentation: Jacket lighting is divided into three zones - hands, pockets, and torso - each controlled independently via MOSFETs.
  • Pattern Control: Pre-programmed lighting sequences include chase, pulse, flash, and sync modes written using the FastLED library.
  • Wireless Trigger: A NodeMCU-based system sends commands wirelessly via 433 MHz RF modules to trigger lighting effects.
  • Multi-Device Sync: Multiple jackets can be triggered simultaneously using a central controller, ensuring synchronized visuals across performers.

Hardware Details

  • Arduino Uno as the primary controller for LED handling and logic execution.
  • ESP8266 NodeMCU board used for remote triggering and communication.
  • Custom power circuit with IRFZ44N MOSFETs for 12V LED switching.
  • Portable 12V battery pack for stage portability and autonomy.

Future Development Goals

  • Music-reactive lighting using FFT audio analysis on-board the controller.
  • Dedicated mobile app for real-time selection of lighting sequences.
  • Custom PCB design for reducing size and increasing reliability of connections.
  • Battery monitoring and overcurrent protection circuitry.

Status

The jacket currently supports standalone and remote-triggered modes for five costumes used in a group dance sequence. Further work is in progress to add interactivity, music responsiveness, and advanced synchronization options. This version represents a functional prototype stage with strong potential for enhancement and public performance scalability.
