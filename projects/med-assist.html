MedAssist - Smart Medical Aid Prototype

Technologies Used

Arduino

ESP8266

C/C++

Pulse Sensor

OLED Display

UART Serial

MedAssist is a proof-of-concept prototype for an embedded health-monitoring platform. It is designed to track basic patient vitals such as heart rate and body temperature using microcontroller-based modules. The aim is to integrate these readings into a wireless system that can transmit data to mobile applications for further analysis, alerting, and remote care support.

Implemented Functionalities

  • Pulse Monitoring: Captures live heart rate data from finger pulse sensor and displays BPM on OLED screen.
  • Temperature Sensing: Reads ambient or body temperature using a digital sensor and logs it periodically.
  • Microcontroller Control: All measurements processed via Arduino Uno and NodeMCU for serial relay.
  • OLED Interface: Provides real-time feedback using a small 0.96" display embedded on the hardware.

Scope of Current Build

This version is a foundational prototype that validates sensor integration, live reading display, and internal data flow. Wireless communication is established at a serial level but not fully integrated with a frontend application or cloud service.

Planned Future Enhancements

  • Bluetooth/Wi-Fi sync with mobile app for real-time health alerts and graphs.
  • Fall detection and movement tracking using accelerometers.
  • Automatic SOS messaging based on medical thresholds.
  • Cloud-based dashboard for doctors and caregivers.

Development Status

Only the basic sensor acquisition and hardware interface components have been implemented as of now. Data transmission pipelines, persistent storage, and smart event detection are part of the roadmap. This project remains in early-stage experimental development and will be iterated further based on test feedback and intended deployment environment.
